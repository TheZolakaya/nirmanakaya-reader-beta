# Nirmanakaya Reader v6

## What's New

### 1. House-Based Card Colors
Cards are now colored by the House of the position:

| House | Color |
|-------|-------|
| Gestalt | Amber |
| Spirit | Violet |
| Mind | Cyan |
| Emotion | Blue |
| Body | Green |
| Portal | Rose |

Status is shown as a smaller badge on the card.

### 2. Intro Section
When landing on the page, users see:
- Brief explanation of what the reader does
- Example questions to inspire
- Disappears once reading is drawn

### 3. Clearer Card Layout
```
┌─────────────────────────┐
│ [Too Much]              │  ← Status badge
│                         │
│ Too Much                │  ← Status prefix
│ CLARITY                 │  ← Transient (large)
│                         │
│ in your Sacrifice       │  ← Context (house-colored)
│ (Position 12)           │
│                         │
│ Cognition Channel       │
│ Expresses Actualization │
│                         │
│ Correction              │
│ → Activation            │
└─────────────────────────┘
```

## Example Questions (shown to users)
- "What do I need to see about this relationship?"
- "Where am I stuck in my work right now?"
- "What's asking for my attention today?"
- "Am I on the right path with this decision?"
